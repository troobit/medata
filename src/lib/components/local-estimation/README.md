# Workstream C: Local Food Estimation Components

**Branch**: `workstream-c/local-food`

Svelte components for browser-local food volume estimation UI.

## Components to create

- `ReferenceCardGuide.svelte` - Guide overlay for card placement
- `FoodRegionSelector.svelte` - Touch/draw food region selection
- `VolumePreview.svelte` - Show estimated volume with uncertainty
- `FoodTypeSelector.svelte` - Search and select food type
- `EstimationResult.svelte` - Final estimation with edit option
